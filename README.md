## ✨ Sobre o projeto

Este projeto foi desenvolvido no **3º ano do ensino médio técnico em TI** como uma atividade integradora entre diferentes disciplinas.

A proposta era criar um **portfólio de TI** para apresentar aprendizados, atividades e conteúdos desenvolvidos durante o ano letivo.

Na página **Áreas do Saber**, cada disciplina é representada por uma atividade escolhida entre as que mais gostei de realizar. O projeto também conta com páginas voltadas para apresentação, tecnologias estudadas e formulário de contato.

---

## 🎯 Objetivo

Criar um portfólio web para reunir conteúdos, atividades e aprendizados adquiridos ao longo do curso técnico.

Além de cumprir a proposta acadêmica, o projeto também serviu como prática de desenvolvimento web, estruturação de páginas, estilização com CSS e uso de interatividade com JavaScript.

---

## 🧩 Funcionalidades

* Página inicial com apresentação
* Página **Áreas do Saber** com atividades das disciplinas
* Página de **Tecnologias** com conteúdos aprendidos
* Navegação entre páginas
* Estrutura visual personalizada
* Formulário de contato com PHP
* Deploy online com GitHub Pages

> Observação: o formulário em PHP não funciona no GitHub Pages, pois esse tipo de hospedagem não executa código back-end.
